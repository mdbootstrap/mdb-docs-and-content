Las máscaras alteran la visibilidad de un elemento ocultándolo parcial o totalmente. Las mascaras son se utiliza para hacer que el contenido sea más visible al proporcionar un contraste adecuado. Se utilizan con mayor frecuencia en imágenes.

Cómo funciona

Una explicación detallada de cómo funcionan las máscaras en MDB:

  • Las máscaras requieren un contenedor .bg-image que establece una posición en relativa, desbordamiento oculto y centre correctamente la imagen.
  • El contenedor interno .bg-image como primer hijo colocamos un img elemento con el enlace de origen
  • A continuación se muestra la máscara real. Establecemos un color y una opacidad a través de Código rgba y CSS en línea. En la sección , encontrará una explicación detallada de cómo funcionan los colores RGBA con las máscaras
  • Si desea poner un texto en la imagen, debe colocarlo dentro del contenedor .mask. A centrarlo tienes que usar utilidades flex (explicadas en el sección a continuación).

RGBA

Al manipular el código RGBA, puede cambiar el color y la opacidad de la máscara.

Incluso puede establecer un degradado elegante como máscara.

rgba(57, 192, 237, 0.6)

Opacidad

Al cambiar el último valor en el color RGBA, puede manipular la opacidad de la máscara.

0.0 significa completamente transparente y 1.0 completamente opaco. Puede establecer cualquier valor entre 0.0 y 1.0.

rgba (0, 0, 0, 0.0) - totalmente transparente

rgba (0, 0, 0, 1.0) - completamente opaco

Contenido

El objetivo principal de las máscaras es proporcionar un contraste adecuado entre la imagen y el contenido. colocado sobre él. Por lo general, colocamos un texto en las imágenes con máscaras.

Dentro del envoltorio .mask coloque un div y aplique utilidades flexbox para centrar el contenido vertical y horizontalmente. Luego ponga el texto adentro.
