Angular Google Maps-API

在本节中，您将找到有关所需模块和表单组件可用输入的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{ButtonsModule，InputsModule}
// MDB Angular从&#39;angular-bootstrap-md&#39;导入{ButtonsModule，InputsModule}
//角度Google Maps从&#39;@ agm / core&#39;导入{AgmCoreModule}
汇入范例模块清单

加 AgmCoreModule.forRoot（{apiKey：&#39;您的钥匙在这里&#39;}） 在导入声明中的app.module.ts文件中以查看地图（获取API密钥这里 ）。
