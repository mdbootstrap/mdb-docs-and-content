上面的命令将运行两个进程-一个负责服务器操作，另一个负责Angular应用程序操作。现在选择一个文件并按Upload按钮就足够了，过一会儿，该文件将在您的项目中的uploads目录中。该文件的名称被散列，因此可以避免重复文件。
