Angular Bootstrap外观是基本的配色方案，可用于提供随时使用的整页网页设计的网站。

MDB为您提供了几种基本的肤色方案，您可能会考虑将其应用到您的网站上以产生不同的效果。

要应用皮肤，您只需要将我们的皮肤类别之一添加到<body>元件。

皮肤会影响以下元素： 导航栏 ， sideNav ，页脚 ，纽扣 ，落下 ，分页 ，输入 ，轮播多项目控件 ，表格标题 ，卡头 ，微调器 ，分页 ，渐变色 ，主要，次要，默认颜色 。
