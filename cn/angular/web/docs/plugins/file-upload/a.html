角度拖放文件上传-API

在本节中，您将找到有关所需模块以及拖放文件上传插件的可用输入，输出，方法和事件的信息。

下载

此插件需要购买。

购买拖放文件上传插件-$ 9

组件

MdbFileUpload

选择器： mdb文件上传

类型： MdbFileUploadComponent

输入

名称 类型 默认 描述 范例
defaultFile 字符串 - 允许将url设置为将用于默认预览的文件 [defaultFile] =“&#39;https://mdbootstrap.com/img/Photos/Others/images/89.jpg&#39;”
禁用 布尔值 false 允许切换禁用状态 [disabled] =“ true”
高 数字 200 允许设置自定义高度 [height] =“ 500”
customText 文字翻译 {fileText：“将文件拖放到此处或单击”，removeText：“删除”，imageFileText：“拖放或单击以替换”} 允许在文件上传组件中设置自定义文本 [customText] =“ customTextTranslation”

输出

名称 类型 描述 范例
fileAdded 事件发射器<File> 添加新文件时触发事件 （fileAdded）=“ onFileAdd（$ event）”
fileRemoved 事件发射器<any> 删除文件时触发事件 （fileRemoved）=“ onFileRemove（）”

方法

您可以从另一个组件访问文件上传方法。将模板引用变量添加到您的 mdb文件上传 HTML文件中的组件

<mdb-file-upload #uploader> </mdb-file-upload>

然后在您的打字稿文件中使用 @ViewChild 装饰者可以访问 MdbFileUploadComponent 方法

ViewChild（&#39;uploader&#39;，{static：true}）上传者：MdbFileUploadComponent

名称 描述 范例
重 置 从输入中删除文件 this.uploader.reset（）
