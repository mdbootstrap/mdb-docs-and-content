角条纹-Bootstrap 4和材料设计

Angular Bootstrap条纹是包含文本的水平块，用于分隔内容部分或突出显示某些信息。

使用它们时应格外小心，因为在一页纸上使用它们太多会使其难以阅读。

条纹可能包含报价或背景图片，其效果类似于视差。
