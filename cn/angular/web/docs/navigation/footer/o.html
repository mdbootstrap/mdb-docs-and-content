角脚-Bootstrap 4和材料设计

Angular Bootstrap Footer是网站的附加导航。它可以包含链接，按钮，公司信息，版权，表格和许多其他元素。

您可以通过添加来自我们的类之一来设置页脚的颜色调色板 。

就像MDBootstrap的任何其他组 件一样，页脚默认情况下也会响应。

基础页脚

添加或删除 .font-small 上课<footer>元素以更改页脚内字体的大小。
