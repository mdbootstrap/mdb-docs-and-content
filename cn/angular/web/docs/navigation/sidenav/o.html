Angular Sidenav-Bootstrap 4和材料设计

Angular Bootstrap sidenav是一个垂直导航组件，允许在小型应用程序和庞大的门户中快速导航。

由于其清晰和简单，它显着提高了用户体验。多重链接嵌入功能使您能够实施更高级的内容分类，这在大型项目中几乎是必不可少的。

借助MDB，您可以通过使用各种诱人的侧边菜单之一轻松地在自己的项目中实现SideNav。

左侧导航是SideNav的实时演示。

    • 固定/非固定SideNav MDB Pro组件

    采用 [fixed] =“ true” 输入和 。固定类以使sidenav固定。该组件将在大屏幕上自动打开，并在小屏幕上隐藏。您可以在左侧看到的导航是固定的SideNav的示例。

    如果您要默认隐藏SideNav，只需删除 。固定类和集合 [固定] 输入到真正 。

    
    <mdb-side-nav #sidenav [fixed] =“ true”>

    需要左填充使用时 。固定类

    如果您打算使用此功能，则必须将内容偏移侧菜单的宽度。将填充物放置在偏移内容将要放置的位置，在我们的示例中是在页眉，主要和页脚中。

    
    页眉，主要，页脚{padding-left：240px; } @media only屏幕和（max-width：992px）{页眉，主要，页脚{padding-left：0; }}

    注意： 无需通过CSS添加所需的填充，您可以使用我们的预定义之一导航布局
