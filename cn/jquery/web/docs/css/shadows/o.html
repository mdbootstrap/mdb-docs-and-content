引导阴影/材料设计阴影

引导阴影是Web元素中使用的一种效果，可提供有关对象深度和方向运动的重要视觉提示。

它们是指示表面之间分离程度的唯一视觉提示。对象的高程确定其阴影的外观。

使用MDB创建阴影非常容易。您只需要将影子类之一添加到所选元素。

示例

将以下类之一添加到任何html元素中，以实现所需的阴影深度

.z-depth-1

.z-depth-1-一半

.z-depth-2

.z-depth-3

.z-depth-4

.z-depth-5

.z-depth-5

Z 深度:

要从给定元素中删除现有阴影，请使用类 .z-depth-0
