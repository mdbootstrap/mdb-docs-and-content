Bootstrap卡片

卡是一种灵活且可扩展的内容容器。它包括页眉和页脚选项，各种内容，上下文背景颜色以及出色的显示选项。

除了通过卡片以外，很难想到一种更好的向用户显示内容的方法。正如您可能已经在几乎所有产品中注意到的那样，一些最大的公司（如Facebook或Google）都非常了解这一点。

卡片为您提供了清晰，干净的内容分类以及向用户展示的吸引人的形式。

MDB通过独特的“材质设计”功能（例如轻微的阴影，深度，级联或波浪效果）增强了Bootstrap卡。除此之外，它还通过其他可选动画，社交图标，化身以及本机Bootstrap中不可用的各种其他效果丰富了它。
