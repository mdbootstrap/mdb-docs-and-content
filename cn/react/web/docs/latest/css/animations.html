滚动时显示动画

借助MDB，您可以轻松地在页面滚动上启动动画。

基本用法

步骤1： 初始化滚动脚本的脚本。

新的WOW（）。init（）;

第2步： 将CSS类.wow添加到HTML元素：在用户滚动显示它之前，它将不可见。

<img src="..." class="wow">

第三步： 从中选择动画样式动画列表 ，然后将CSS类添加到HTML元素。

<img src="..." class="wow fadeInUp">

可见性：隐藏；

加可见度：隐藏到哇如果您要为页面加载后立即可见的元素设置动画。因此，您避免在javascript加载后重新加载内容。

.wow {能见度：隐藏； }

选项

使用下面的自定义属性之一来更改滚动上动画的行为。

  1. 数据持续时间 ：更改动画持续时间

  2. 数据哇延迟 ：动画开始之前的延迟

  3. 数据哇偏移 ：开始动画的距离（与浏览器底部有关）

  4. 数据哇重复 ：动画重复的次数

<img src="..." class="wow fadeInUp" data-wow-delay="0.6s">

自定义设置

  1. boxClass ：类名称，在用户滚动时显示隐藏的框

  2. 动画类 ：触发CSS动画的类名

  3. 抵销 ：定义浏览器视口底部与隐藏框顶部之间的距离。当用户滚动并到达此距离时，将显示隐藏的框。

  4. 移动 ：打开/关闭移动设备上的动画。

  5. 生活 ：不断检查页面上是否有新的动画元素

wow = new WOW（{boxClass：&#39;wow&#39;，//默认animateClass：&#39;animated&#39;，//默认偏移量：0，//默认移动设备：true，//默认live：true //默认}）wow.init（ ）;
