启用/禁用分页

分页默认情况下处于启用状态，因此，如果您想要分页，则无需执行任何操作。

通过在Datatable组件中设置属性，可以轻松地禁用不希望用于特定表的功能。可用选项的完整列表位于 API 此页面的部分。
