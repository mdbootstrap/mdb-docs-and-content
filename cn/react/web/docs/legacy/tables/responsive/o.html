Vue Bootstrap表响应

响应式表允许轻松地水平滚动表。通过添加属性使任何表在所有视口中都具有响应性反应灵敏 。或者，使用以下命令选择一个最大的断点，以使其具有响应表响应式{Sm | Md | Lg | Xl} 。

垂直裁剪/截断

响应表利用溢出y：隐藏 ，它会剪裁超出表格底部或顶部边缘的所有内容。特别是，这可以剪切下拉菜单和其他第三方小部件。

始终响应

在每个断点处使用道具反应灵敏用于水平滚动表。

引导表列宽

使用以下属性之一来操纵列的宽度。

具有自动宽度的表格列

添加道具 autoWidth 到 tbl 元素以设置表格列的自动宽度。

列的宽度将自动调整为列的内容。这意味着它将始终采用呈现内容所需的最小宽度。

具有最小宽度的表列

您必须添加 .th-lg 要么短信上课<th>内的元素<tbl-head> 。

调整浏览器窗口的大小以查看效果。
