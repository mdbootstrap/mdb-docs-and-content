React Bootstrap砌体

React Masonry-Bootstrap 4和材料设计

React Bootstrap砌体是基于列的网格布局。与其他网格布局不同，它没有固定高度的行。基本上，Masonry布局通过减少不必要的间隙来优化网页内部空间的使用。没有这种类型的布局，就需要某些限制来维持布局的结构。
