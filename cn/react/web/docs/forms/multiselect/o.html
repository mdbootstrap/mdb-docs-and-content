React Bootstrap多选 MDB Pro组件

React Multiselect-Bootstrap 4和材料设计。

Bootstrap材质multiselect是一个表单控件，单击后会显示可折叠的多个值列表，可在表单，菜单或调查中使用。

多重选择使您可以使用 ↑ 和 ↓ 箭头键浏览选项并使用 ↵ 键选择所需的选项。
