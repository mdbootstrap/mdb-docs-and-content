反应悬停效果-API

在本节中，您将找到有关MDB中悬停效果的高级信息。

导入账单

为了使用悬停效果，请确保您首先导入了正确的模块。

从“ mdbreact”导入{MDBView，MDBMask};

API参考：视图属性

下表显示了MDBView组件的配置选项。

名称 类型 默认 描述 范例
层叠 真假值 false 放置内容级联 <MDBView级联/>
类名 字符串 自定义类别 <MDBView className="customClass" />
悬停 真假值 false 开启悬停效果 <MDBView悬停/>
圆形的 真假值 false 将View的边界半径设置为50％ <MDBView取整/>
src 字符串 设置固定图像背景；接受图片的网址 <MDBView src="/assets/image.jpg" />
标签 字符串 div 更改默认标签 <MDBView tag="span" />
波 真假值 false 打开波浪效果（鼠标向上） <MDBView波/>
缩放 真假值 false 开启zomm效果（悬停时） <MDBView zoom />

API参考：掩码属性

下表显示了MDBMask组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBMask className="customClass" />
覆盖 字符串 以传递的颜色创建叠加层；颜色列表检查面具的文档 <MDBMask overlay="grey-light" />
调色板 字串号 用选定的样式创建图案；模式列表检查面具的文档 <MDBMask pattern="1" />
标签 字符串 div 更改默认标签 <MDBMask tag="span" />
