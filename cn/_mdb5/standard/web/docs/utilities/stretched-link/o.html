通过CSS“拉伸”嵌套链接，使任何HTML元素或MDB组件都可单击。

基本例子

将 .stretched-link 添加到链接以使其 包含区块 可通过 :: after 伪元素单击。在大多数情况下，这意味着 带有 position：relative; 的链接，其中包含与 .stretched-link 类是可单击的。

卡在Bootstrap中默认具有 position：relative ，因此在这种情况下，您可以 安全地将 .stretched-link 类添加到卡中的链接，而无需任何其他HTML 变化。

拉伸链接不建议使用多个链接和点击目标。但是，有些 如果需要， position 和 z-index 样式可以提供帮助。

识别包含块

如果拉伸后的链接似乎无效，则 包含区块 可能是原因。以下CSS属性将使元素包含 块：

  • 除了 static 之外的 position 值
  • 除 none 以外的 transform 或 perspective 值
  • transform 或 perspective 的 will-change 值
  • 除 none 之外的 filter 值或 will-change 值 （过滤器）（仅适用于Firefox）
