单选按钮是用于允许用户在多个 选项（而 复选框用于选择多个 选项）。

基本例子

浏览器的默认复选框和单选按钮已被替换为 .form-check ，这两种输入类型的一系列类，可改善布局 和HTML元素的行为，从而提供更大的自定义和跨浏览器 一致性。复选框用于选择列表中的一个或多个选项，而收音机则用于 从众多选项中选择一种。

从结构上讲，我们的< input > s和< label > s是同级的 与< label > 中的< input > 相对的元素。这个 较为冗长，因为您必须为 属性指定 id 和 关联 for < input > 和< label > 。

我们将兄弟选择器（〜）用于所有 < input > 状态，例如：checked 或：disabled 。什么时候 结合 .form-check-label 类，我们可以轻松地为每个文本设置样式 基于< input > 的状态的项目。

残障人士

添加 disabled 属性，并且关联的< label > 是 自动设置样式以与较浅的颜色匹配，以帮助指示输入状态。

排队

通过添加将同一水平行上的复选框或单选按钮分组 .form-check-inline 到任何 .form-check 。

没有标签

对于没有标签文本的复选框和单选框，请省略包装的 .form-check 。 请记住，仍要为辅助技术提供某种形式的标签（例如，使用 aria-label ）。

切换按钮

Toggle buttons can be grouped in a button group if needed.
