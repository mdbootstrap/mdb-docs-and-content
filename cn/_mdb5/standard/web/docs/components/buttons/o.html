色彩

MDB包含几种预定义的按钮样式，每种样式都有其自己的语义目的，并且 很少有其他东西可以控制。

传达辅助技术的意义：
使用颜色添加含义仅提供视觉指示，不会传达给 辅助技术的用户-例如屏幕阅读器。确保信息表示为 颜色可以从内容本身（例如可见文字）中显而易见，也可以包含在内 通过其他方式，例如用 .sr-only 类。

禁用状态

通过将 disabled 布尔属性添加到任何按钮，使按钮看起来不活动 <button> 元素。禁用的按钮有 pointer-events：不适用，以防止发生悬停状态和活动状态 触发。

使用 元素的禁用按钮的行为略有不同：

  • 不支持 disabled 属性，因此您必须添加 .disabled 类以使其在外观上显示为禁用。
  • 包括一些未来友好的样式以禁用所有 锚按钮上的 pointer-events 。在支持该属性的浏览器中， 根本看不到禁用的光标。
  • 禁用的按钮应包括 aria-disabled =" true" 属性指示元素的状态 辅助技术。

链接功能警告：
.disabled 类使用 pointer-events：none 尝试禁用 的链接功能，但是CSS属性尚未标准化。 另外，即使在支持 pointer-events：none ，键盘的浏览器中 导航不受影响，这意味着有视力的键盘用户和辅助用户 技术仍将能够激活这些链接。为了安全起见，请添加一个 这些链接上的 tabindex ="-1" 属性（以防止它们接收键盘 焦点）并使用自定义JavaScript禁用其功能。
