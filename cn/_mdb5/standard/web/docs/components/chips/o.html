芯片-Bootstrap 5和Material Design 2.0组件

Bootstrap标签和芯片使用文本和图标对内容进行分类。标签和芯片 使浏览整个文章，评论或页面更加容易。他们的主要目标是提供 您的访客以直观的方式获得他们想要的东西。考虑一下，有多方便 只需使用标签即可找到与Web开发相关的所有文章。

注意：阅读 API 标签以找到所有可用选项，然后 高级定制

占位符

输入名称，然后按Enter键添加标签。单击X将其删除。

初始值

您可以使用js选项设置初始标签。

 const chipsInitialNew = document.querySelector('.chips-test') const newChipsNew = new mdb.ChipsInput(chipsInitialNew, { initialValues: [{ tag: 'MDBReact' }, { tag: 'MDBAngular' }, { tag: 'MDBVue' }, { tag: 'MDB5' }, { tag: 'MDB' }] })

内容可编辑

您可以设置内容可编辑，以将选项 editable 添加为true（使用JavaScript或 data-mdb属性。

 const chipsInitial = document.querySelector('.chips-test') const newChips = new mdb.ChipsInput(chipsInitial, { editable: true })
