基本例子

工作原理

进度组件由两个HTML元素构建，一些CSS设置宽度，另一些 属性。我们不使用 HTML5 <进展> 元件，以确保您可以堆叠进度条，为其设置动画并在其上放置文本标签。

  • 我们使用 .progress 作为包装器来指示进度的最大值 酒吧。
  • 我们使用内部的 .progress-bar 指示到目前为止的进度。
  • .progress-bar 需要内联样式，实用程序类或自定义CSS进行设置 他们的宽度。
  • .progress-bar 还需要一些 role 和 aria 属性以使其可访问。

将它们放在一起，您将看到以下示例。

MDB提供了一些 用于设置宽度的实用程序。取决于你 需求，这些可能有助于快速配置进度。

高度

我们只在 .progress 上设置一个 height 值，所以如果您更改该值 内部 .progress-bar 的值将自动相应地调整大小。
