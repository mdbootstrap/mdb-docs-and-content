MDB Pro 组件

存储组件使您可以管理存储在浏览器内存中的数据。感谢 组件，您可以轻松地添加，删除，获取数据并检查其过期时间。

注意:阅读 API 标签以找到所有可用选项，并 高级定制

基本例子

存储组件提供添加，删除和获取存储的方法。

设置存储

使用方法 Storage.set（）将数据添加到存储中。您可以测试此方法 使用以下示例。该按钮将调用 Storage.set（'date'，new Date（））; 。要再次使用此示例，请按重置 按钮。

Storage.set('date', new Date());

设置到期时间

您可以设置本地存储中已保存数据的到期时间（以天为单位）

Storage.set('date', new Date(), 1);

获取存储空间

如果您已将任何数据保存在存储器中，则可以使用 Storage.get 方法。

Storage.get('name');

删除存储

当不再需要保存在存储器中的数据并且想要删除它们时，请使用 Storage.remove 方法。

Storage.remove('name');

检查存储

您可以设置一个任务来检查数据是否已过期，删除它并设置回调函数。 设置数据名称，间隔 interval （以分钟为单位），以检查频率 到期日期和回调fn 。

Storage.check('name', 0.5, () => { // do something });
