MDB Pro component

虚拟插件可创建占位符内容，以加速构建和测试前端 模板。

注意：阅读 API 标签以找到所有可用选项，然后 高级定制

通过向其添加类 dummy 为该元素创建Dummy组件。取决于 基本元素，将创建不同的假人。对于大多数HTML标签，默认Dummy为 文本。

要显式创建特定的Dummy，请使用以下命令向元素添加适当的 data-mdb-属性 dummy 类

文本

通过向元素添加 data-mdb-dummy-text 来创建虚拟文本。用创建文本 自定义长度将所需长度的值添加到属性。使用更改文字颜色 data-mdb-text-color 属性。

图片

通过将 data-mdb-dummy-img 添加到元素来创建虚拟图像。

默认

默认情况下，虚拟图像的大小为150x150px，背景为灰色

尺寸

使用 data-mdb-dummy-img =“宽度值，高度值” 更改虚拟图像的大小。 高度值是可选的，省略时将创建正方形图像。

使虚拟图像继承其容器的完整大小 data-mdb-dummy-img =“ 100％，100％” 。

注意：仅适用于具有明确设置宽度和宽度的容器 高度。

设置 data-mdb-dummy-img =“ 100％” 将创建宽度为 容器宽度的高度

文本

使用 data-mdb-dummy-img-text 属性将文本添加到虚拟图像中。传递任何值 设置自己的文本或留空以在图像内部创建虚拟文本。组 带有 data-mdb-dummy-text-length =“ text length value” 的虚拟文本的长度

色彩

使用 data-mdb-color 设置图像背景颜色，并使用 data-mdb-dummy-text-color 。虚拟图像仅接受十六进制的颜色值

清单

通过将 data-mdb-dummy-list 添加到 ul 来创建虚拟列表 ol 元素。

默认

默认情况下，虚拟列表将创建包含5个虚拟文本元素的列表

      尺寸

      使用 data-mdb-dummy-list =“长度值” 更改虚拟列表的长度和 具有 data-mdb-text-length =“文本长度值” 的虚拟文本元素。

          表

          通过将 data-mdb-dummy-table 添加到 table 元素来创建虚拟表。

          默认

          默认情况下，虚拟表将创建5行5列的表。

          尺寸

          更改虚拟表的行和列大小 data-mdb-dummy-table =“行长度值，列长度值” 。变更长度 具有 data-mdb-text-length =“ text length value” 的默认虚拟文本

          复制

          使用 data-mdb-dummy-copy =“现有元素选择器” 创建任何现有元素的虚拟副本。

          模板

          虚拟模板将创建单个虚拟元素或标签元素中的整个模板 作为字符串传递到 data-mdb-dummy-template 中。

          默认

          默认情况下，虚拟模板将创建以下模板 h1，p，table，h2，p，form，blockquote，img，ul 标签。

          版面

          将虚拟模板与其他虚拟元素和MDBootstrap组件结合起来创建 立即布局！
