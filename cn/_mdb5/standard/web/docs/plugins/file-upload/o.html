MDB Pro 组件

MD Bootstrap's File Upload plugin is an extension that allows you to upload files by using drag and drop functionality. Easy to use, setup and customize.

注意：阅读 API 标签以找到所有可用选项，并 高级定制

基本例子

For each input with a data-file-upload="file-upload" attribute you have to add a wrapper with the class file-upload-wrapper.

Default message example

By adding data-default-msg attribute you can set main message of the file upload.

Custom height example

By adding data-height attribute you can set its custom height.

Max size

By adding data-max-file-size attribute you can set max size of a file.

Default value

By adding data-default-value attribute you can set default file in the file upload element.

Disable

By adding data-disabled attribute you can disable the component.

Accept formats

By adding data-accepted-formats you can set allowed file types.

Disabled remove button

By adding data-disabled-remove-btn attribute you can remove "Remove button".

Multiple files

By adding data-multiple attribute you can allow to upload more than single file.

Multiple with files limit

By adding data-max-file-quantity attribute you can set limit of uploaded files.
