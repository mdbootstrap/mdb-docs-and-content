MDB Pro 组件

MD Bootstrap的"文件上传"插件是一个扩展，可让您通过使用 拖放功能。易于使用，设置和自定义。

注意：阅读 API 标签以找到所有可用选项，并 高级定制

基本例子

对于每个具有 data-file-upload =" file-upload" 属性的输入，您必须添加一个 类为 file-upload-wrapper 的包装器。

默认消息示例

通过添加 data-default-msg 属性，您可以设置文件上传的主要消息。

自定义高度示例

通过添加 data-height 属性，您可以设置其自定义高度。

最大尺寸

通过添加 data-max-file-size 属性，您可以设置文件的最大大小。

默认值

通过添加 data-default-value 属性，您可以在文件中设置默认文件 上传元素。

禁用

通过添加 data-disabled 属性，您可以禁用该组件。

接受格式

通过添加数据接受的格式，您可以设置允许的文件类型。

禁用删除按钮

通过添加 data-disabled-remove-btn 属性，您可以删除"删除按钮"。

多个文件

通过添加 data-multiple 属性，您可以上传多个文件。

多个文件限制

通过添加 data-max-file-quantity 属性，您可以设置上传文件的限制。
