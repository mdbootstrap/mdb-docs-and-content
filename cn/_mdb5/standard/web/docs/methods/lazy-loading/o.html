Bootstrap 5延迟加载是一项功能，它允许您仅在图像或视频时才加载它们 在屏幕上可见。

注意:阅读 API 标签以找到所有可用选项，并 高级定制

基本例子

将 .lazy 类添加到您的计算机后，延迟加载将自动初始化 img 或 video 元素。重要的是要添加 data-lazy-src 属性-否则，延迟加载将引发错误。

抵消

使用 data-lazy-offset 属性在之后定义额外的像素数量 将显示哪个图像或视频。

向下滚动以加载图片。

错误

使用 data-lazy-error 定义图片，如果图片或视频不显示，则将显示该图片 加载。

动画制作

使用 data-lazy-animation 指定要在激活时激活的动画 单元载荷。

容器

通过将 .lazy 类添加到其容器中，可以一次初始化一组元素。如果 您可以在父级，所有 img 的父级中设置 data-lazy 属性 里面的 video 元素将继承它们。您可以通过声明覆盖它们 这些属性在内部元素中。
