基本例子

添加时会自动初始化滚动条 容器的 data-perfect-scrollbar ='true'属性。

选件

JavaScript初始化

您可以使用JavaScript轻松启动滚动条。你必须调用 mdb.PerfectScrollbar（）并在其中包含选项。

const initByJS = document.querySelector('#init-by-js'); const psInitbyJS = new mdb.PerfectScrollbar(initByJS);

资料属性

您可以轻松地使用包含数据属性的选项来初始化滚动条。您必须添加 data-perefect-scrollbar 到包装器。如果要使用data-attr添加选项 您必须在币库中添加例如 data-suppress-scroll-x ='true'。 。

大事记

PerfectScrollbar调度自定义事件。

  • ScrollY
  • ScrollUp
  • ScrollDown
  • ScrollLeft
  • ScrollRight
  • ScrollXEnd
  • ScrollYEnd
  • ScrollXStart
  • ScrollYStart
