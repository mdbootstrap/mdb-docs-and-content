MDB Pro 组件

粘性是一个组件，它可以将元素锁定在页面的特定区域中。它 通常在导航菜单中使用。

注意：阅读 API 标签以找到所有可用选项，并 高级定制

基本例子

要开始使用sticky，只需在要固定的元素上添加一个 .sticky 类

粘底

您可以通过添加将元素固定到底部 data-sticky-position =“ bottom”

边界

设置 data-sticky-boundary =“ true” ，以便粘性仅在父级内部起作用 元件。记住要设置正确的父母身高。

方向

粘性组件的默认方向是 down 。您可以通过设置进行更改 data-sticky-direction =“ up” 或 data-sticky-direction =“ both”

动画

您可以添加一个动画，该动画将在粘滞开始时以及粘滞元素为 隐。只需使用指定动画的css类 data-sticky-animation-sticky 和 data-sticky-animation-unsticky 属性。
