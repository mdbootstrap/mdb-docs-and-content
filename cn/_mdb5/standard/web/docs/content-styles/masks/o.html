怎么运行的

掩码如何在MDB中工作的详细说明：

  • 遮罩需要 .bg-image 包装器，该包装器将位置设置为相对，溢出到 隐藏并正确居中图像。
  • 内部的 .bg-image 包装器作为第一个子容器， 带有源链接的 img 元素
  • 下面是实际的蒙版。我们通过设置颜色和不透明度 rgba 代码和内联CSS。在部分中，您会找到一个 RGBA颜色如何与遮罩配合使用的详细说明
  • 如果要在图像上放置文本，则必须将其放在.mask包装器中。至 居中，您必须使用flex实用程序（在 部分）。

不透明度

通过更改RGBA颜色中的最后一个值，您可以控制蒙版的不透明度。

0.0表示完全透明，而1.0表示完全不透明。您可以在0.0到1.0之间设置任何值。

rgba(0, 0, 0, 0.0) - fully transparent

rgba(0, 0, 0, 1.0) - fully opaque

内容

蒙版的主要目的是在图像和内容之间提供适当的对比度 放在上面。最常见的是，我们在带有蒙版的图像上放置文本。

在 .mask 包装器中放置一个 div 并应用 flexbox实用程序以使内容居中 垂直和水平。然后将文字放入其中。
