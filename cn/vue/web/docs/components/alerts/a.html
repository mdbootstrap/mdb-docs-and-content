API参考：警报事件

名称 描述 范例
进入之前 当警报即将出现时，将发出此事件。 <mdb-alert @beforeEnter = {（）=> alert（&#39;beforeEnter event&#39;）}>
进入 当出现警报时，将发生此事件。 <mdb-alert @enter = {（）=> alert（&#39;输入事件&#39;）} >>
之后进入 警报到位后，将发生此事件。 <mdb-alert @afterEnter = {（）=> alert（&#39;afterEnter event&#39;）}>
输入已取消 如果转换已中断，则会发生此事件。 <mdb-alert @enterCancelled = {（）=> alert（&#39;enterCancelled事件&#39;）}>
离开之前 当警报即将消失时，将发出此事件。 <mdb-alert @beforeLeave = {（）=> alert（&#39;beforeLeave事件&#39;）}>
离开 当警报开始消失时，将发生此事件。 <mdb-alert @leave = {（）=> alert（&#39;离开事件&#39;）}>
之后离开 警报消失后，将发生此事件。 <mdb-alert @afterLeave = {（）=> alert（&#39;afterLeave事件&#39;）}>
取消 如果离开过渡已中断，则会发生此事件。 <mdb-alert @leaveCancelled = {（）=> alert（&#39;leaveCancelled事件&#39;）}>
