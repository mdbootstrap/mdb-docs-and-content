手风琴

名称 类型 默认 描述
窗格 对象/数组 空 用于自动生成窗格的数据数组。其中的每个对象都应有一个标题 ，内容以及可选图标属性
打开： 号码 空 MDB Vue手风琴开始时会关闭窗格。要更改此行为，请添加道具以指定要在初始渲染中打开的数量。记得 -此处的计数从 0 。由于您要为组件提供数字道具，因此请记住通过添加冒号来尊重其道具类型
默认 布尔值 如果为 true，则幻灯片将没有固定位置 引导手风琴风格
材料 布尔值 false 符合材料设计标准的手风琴
图片 布尔值 false 涉及浅蓝色标题的样式
图标 布尔值 false 具有这种风格的手风琴应具有白色背景并使用 ：窗格数据数组对象图标血红色的财产
颜色 字符串 空 该道具将与我们的“五色窗格”手风琴挂钩，您可以接受我们的任何颜色
相片 布尔值 false 它利用了照片背景手风琴的典型样式
表 布尔值 false 帮助设置基于表的高度可读的手风琴组件
标签 字符串 div 允许更改组件的本机HTML标签。

手风琴潘

如果您希望自己的窗格自定义，这是用于显示手风琴内容的组件的API。请记住-这样操作会给您带来操作逻辑的重担！参考桌子手风琴的例子查看应该为手风琴工作提供的方法和数据对象。
