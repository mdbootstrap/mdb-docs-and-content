折叠-API

在本节中，您将找到有关“折叠”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbCollapse};

API参考：属性

名称 类型 默认 描述 范例
拨动开关 号码 1 切换按钮的数量。 <mdb-collapse：togglers =“ 2” ... >
toggleTag 字符串数组 按钮 切换标签 <mdb-collapse：toggleTag =“ [&#39;a&#39;，&#39;button&#39;]” ... >
toggleText 字符串数组 切换 确定每个切换器的内部文本 <mdb-collapse：toggleText =“ [&#39;折叠链接&#39;，&#39;折叠按钮&#39;]” ... >
toggleClass 字符串数组 [&#39;btn btn-primary&#39;] 确定每个切换器的类 <mdb-collapse：toggleClass =“ [&#39;折叠链接&#39;，&#39;折叠按钮&#39;]” ... >
