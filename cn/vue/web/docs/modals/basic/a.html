名称 类型 默认 描述

|||UNTRANSLATED_CONTENT_START|||position|||UNTRANSLATED_CONTENT_END||| 字符串 -- 确定模态相对于浏览器窗口的位置

fullHeight 真假值 false 使模态从指定边开始全高

removeBackdrop 真假值 false 沿其功能从背景上去除深色的覆盖层
层叠 真假值 false 允许模态的级联性质
头像 真假值 false 通过基于化身的标题完美呈现模态样式
选项卡 真假值 false 提供模式标题中具有标签导航所需的样式
成功 真假值 false 以成功通知的方式模态模式
info 真假值 false 信息通知方式的样式模式

警告 真假值 false 以警告通知方式显示样式模式
优雅 真假值 false 它有助于样式情调的样式，使其既光滑又优雅
黑暗 真假值 false 确定模态的样式是否不应该具有黑暗性质
bgSrc 字符串 空 深色样式必须具有图片背景，但实际上该功能适用于所有模态
可滚动 真假值 false 允许使用滚动条滚动浏览长内容
方向 字符串 "TOP": { 允许更改转场的方向。
