下表显示了导航栏零件。

名称 类型 默认 描述 范例
标签 字符串 纳芙卡 设置组件的标签 <mdb-navbar tag="div" />
动画 字串号 <mdb-navbar animation="1" />

黑暗 真假值 false 调整字体颜色以与较深的导航栏颜色形成对比 <mdb-navbar dark />

容器 真假值 false 在navbar的内容中添加一个容器包装器 <mdb-navbar容器/>
双线 真假值 false SideNav使导航栏成为了一项壮举 <mdb-navbar double />
展开 字符串 小 确定应在哪个视口上导航栏展开。接受： 短信 / 小 ， md / 介质 ， lg / 大 。如果为空-导航栏将不会折叠。 <mdb-navbar expand="md" />
navIconClass 字符串 -- 设置切换器CSS类 <mdb-navbar position="top" />
|||UNTRANSLATED_CONTENT_START|||position|||UNTRANSLATED_CONTENT_END||| 字符串 设置道具会使导航栏固定。赋值是 “最佳” 和 “底部” <mdb-navbar position="top" />
滚动 真假值 false 在滚动条上更改导航栏的大小。与道具结合透明达到变色效果 <mdb-navbar滚动/>
scrollingOffset 号码 100 表示垂直滚动的像素数，在该像素上方应用“滚动”效果 <mdb-navbar :scrollingOffset={100} />
togglerClass 字符串 -- 有助于在汉堡包上设置自定义样式 <mdb-navbar :scrollingOffset={100} />
透明 真假值 false 将导航栏背景设置为透明。与道具结合卷动和颜色达到变色效果的道具 <mdb-navbar透明/>
切换 真假值 如果为 true，则幻灯片将没有固定位置 调成假防止导航栏崩溃到汉堡菜单 <mdb-navbar :toggler="false" />
汉堡 真假值 调成真正启用设置自定义切换图标 <mdb-navbar汉堡包/>

togglerSize 字符串 1x 更改切换图标的大小 <mdb-navbar汉堡包togglerIcon =“ heart” togglerSize =“ 2x” />

Fal 真假值 false 将切换图标的样式更改为浅色 <mdb-navbar汉堡包togglerIcon =“ ...” fal />
很棒 真假值 false 将切换图标的样式更改为品牌 <mdb-navbar汉堡包togglerIcon =“ ...” fab />

API参考：mdbNavItem属性

下表显示了 mdb-navbar-nav 零件。

名称 类型 默认 描述 范例
标签 字符串 li 设置组件的标签 <mdb-navbar-brand tag="div" />
活动 真假值 false 将NavItem的状态设置为活动 <mdb-nav-item :active="chechIfActive" />
禁用 真假值 false 禁用元素链接 <mdb-nav-item :disabled="AreYouDisabled" />
精确 真假值 false 主动式类包括在内。使用道具使组件变为活动状态只要当href道具与$ route.path完全匹配时。可以用来路由器仅支撑的组件 <mdb-nav-item href="/" exact="AreYouDisabled" />
href 字符串 -- 点击应导致的外部URL <mdb-nav-item href="http://www.yeah.why.not.here" />
新标签 真假值 false 用于在新的浏览器选项卡中打开页面。等效于ol&#39;target =“ _ blank”锚元素属性 <mdb-nav-item :waves="false" />
到 字符串 -- 使用这个道具代替 href 使组件呈现一个路由器链接 。利用它在SPA项目中导航。 <mdb-nav-item :waves="false" />
波 真假值 false 单击组件时设置波浪效果。采用固定代替固定元素 <mdb-nav-item :waves="false" />
图标 字符串 在导航项旁边添加图标 <mdb-nav-item icon="code-branch" />
較遠的 / 较远的 (jiào yuǎn de) 真假值 false 两种道具都将图标的风格颂唱成常规 <mdb-nav-item icon="..." far />
Fal 真假值 false 两种道具都彰显着图标的风格 <mdb-nav-item icon="..." fal />
很棒 真假值 false 这两个道具都在向品牌传达图标的风格 <mdb-nav-item icon="..." fab />
時尚潮流 真假值 false 两种道具都向偶像颂唱偶像风格 <mdb-nav-item icon="..." fad />
anchorClass 字符串 将课程添加到锚元件 <mdb-nav-item anchorClass="custom-class" />
