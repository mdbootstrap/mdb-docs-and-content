启用/禁用搜索字段

默认情况下，搜索处于启用状态，因此，如果要搜索，则无需执行任何操作。

通过设置属性，可以轻松地禁用不希望用于特定表的功能 ：searching =“ false” 在里面数据表零件。可用选项的完整列表位于数据表参考 。
