Angular Bootstrap e-commerce components

Angular E-commerce components - Bootstrap 4 & Material Design

Angular Bootstrap e-commerce components is a set of components which are necessary to build online stores and any other e-commerce businesses.

Required styles import

Starting with MDB Angular 8, using the following components requires importing an additional stylesheet. Add the following imports to the global styles.scss stylesheet file.

@import "~ng-uikit-pro-standard/assets/scss/core/colors"; @import "~ng-uikit-pro-standard/assets/scss/core/variables"; @import "~ng-uikit-pro-standard/assets/scss/core/variables-pro"; @import "~ng-uikit-pro-standard/assets/scss/addons-pro/_ecommerce-pro.scss";
