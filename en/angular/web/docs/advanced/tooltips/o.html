Angular Bootstrap Tooltips

Angular tooltips - Bootstrap 4 & Material Design

Tooltips are a convenient way of presenting additional information to your user. They are tiny little clouds with a brief text message, triggered by clicking on a specified element or hovering over it.

Custom HTML

You can use HTML to customize a tooltip's title.

 import { Component} from '@angular/core'; @Component({ selector: 'tooltip-dynamic-example', templateUrl: 'tooltip-dynamic.html', }) export class TooltipDynamicComponent { html = 'Tooltip with HTML'; }

Show and Hide

You can use the show() and hide() methods to manually toggle the tooltip. Remember to pass the $event argument to the show() function.

Modify tooltip height

Add .tooltip-inner { height: desired-value } to the styles.scss stylesheet to change the height of the tooltip.

Also, remember to use a customHeight input. This is required if the tooltip has no space to display along the top or bottom edge of the screen, and is displayed the other way round.

.tooltip-inner { height: 50px; line-height: 50px; }
