The ripple method provides a radial action in the form of a visual ripple expanding outward from the userâ€™s touch. Ripple is a visual form of feedback for touch events providing users a clear signal that an element is being touched.

Note: Read the API tab to find all available options and advanced customization

Basic example

By default, ripple is added to every button (via .btn class) and it does not require any additional classes or attributes.

It's also automatically added to the .ripple class. Thanks to this you can use the .ripple class to apply the ripple effect to the given clickable element - for example to the image wrapped in <a> element.

Colors

By using data-ripple-color attribute you can change the color of the ripple.

You can use the colors from the basic MDB palette, for example data-ripple-color="primary" or data-ripple-color="danger":

You can also use any CSS color name:

Or simply use the hex color code:

Duration

By using data-ripple-duration attribute you can modify the duration of the ripple.

Centered

If you add data-ripple-centered="true" attribute the ripple always starts in the center of the element instead of in the touched place.

Unbound

If you use data-ripple-unbound="true" attribute,the ripple won't be bonded to the given element and it will exceed its borders.

Radius

By using data-ripple-radius attribute you can modify the radius of the ripple. The numeric value is expressed in pixels, for example: data-ripple-radius="10"

JavaScript implementation
