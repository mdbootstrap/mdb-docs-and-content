Parallax is a plugin that adds scrolling effect animation for your images.

Note: Read the API tab to find all available options and advanced customization

Basic example

To initialize Parallax effect simply add parallax class to any element and provide data-mdb-image-src attribute with path to your image

Direction

Change parallax scroll direction with data-mdb-direction attribute

Combine horizontal and vertical directions

Delay

Change delay of the scroll with data-mdb-delay attribute

Scale

Change scale of the image with data-mdb-scale attribute

Scroll transition

Change default scroll transition timing function with data-mdb-transition

Max transition

Change maximum scroll value of the image with data-mdb-max-transiton attribute. Give values within 0-100 range

Max image height

Set maximal parallax container height with data-mdb-max-height

Image alignment

Horizontal and vertical alignment of the image is set to center by default. Change horizontal alignment with data-mdb-horizontal-alignment and vertical with data-mdb-vertical-alignment (vertical alignment will only work with data-mdb-max-height set)

Image horizontally aligned

Image vertically aligned

Container overflow

Make parallax overflow its container with data-mdb-overflow

Parallax with content

Place any content inside the parallax by adding div with class parallax-content and your content inside parallax container
