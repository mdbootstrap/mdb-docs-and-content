- API

Methods

You can create a button instance with the button constructor, for example:

var button = document.getElementById('myButton') var bsButton = new mdb.Button(button)
Method Description Example
toggle Toggles push state. Gives the button the appearance that it has been activated. instance.toggle()
dispose Destroys an element's button. instance.dispose()

For example, to toggle all buttons

var buttons = document.querySelectorAll('.btn') buttons.forEach(function (button) { var button = new mdb.Button(button) button.toggle() })

Import

MDB UI KIT also works with module bundlers. Use the following code to import this component:

import { Button } from 'mdb-ui-kit';
