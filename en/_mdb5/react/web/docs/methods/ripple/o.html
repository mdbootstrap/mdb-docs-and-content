The ripple method provides a radial action in the form of a visual ripple expanding outward from the userâ€™s touch. Ripple is a visual form of feedback for touch events providing users a clear signal that an element is being touched.

Note: Read the API tab to find all available options and advanced customization

Basic example

By default, ripple is added to every button and it does not require any additional classes or attributes.

import React from 'react'; import { MDBBtn } from 'mdb-react-ui-kit'; export default function App() { return ( Primary ); }

Wrap other elements into MDBRipple to initialize a ripple effect on them.

import React from 'react'; import { MDBRipple } from 'mdb-react-ui-kit'; export default function App() { return ( ); }

Colors

By using rippleColor property you can change the color of the ripple.

You can use the colors from the basic MDB palette, for example rippleColor="primary" or rippleColor="danger":

import React from 'react'; import { MDBBtn } from 'mdb-react-ui-kit'; export default function App() { return ( <> Primary Secondary Success Danger Info Light Dark ); }

You can also use any CSS color name:

import React from 'react'; import { MDBBtn } from 'mdb-react-ui-kit'; export default function App() { return ( <> Red Green Blue Yellow Orange Purple Aqua ); }

Or simply use the hex color code:

import React from 'react'; import { MDBBtn } from 'mdb-react-ui-kit'; export default function App() { return ( <> #c953d6 #44c6e3 #fcc834 #386f06 #c1303a #a57c3e #192ced #525740 ); }

Duration

By using rippleDuration property you can modify the duration of the ripple.

import React from 'react'; import { MDBBtn } from 'mdb-react-ui-kit'; export default function App() { return ( <> Default (500ms) 1s 3s 5s 10s ); }

Centered

If you add rippleCentered property the ripple always starts in the center of the element instead of in the touched place.

import React from 'react'; import { MDBBtn } from 'mdb-react-ui-kit'; export default function App() { return ( Centered ripple ); }

Unbound

If you use rippleUnbound property,the ripple won't be bonded to the given element and it will exceed its borders.

import React from 'react'; import { MDBBtn } from 'mdb-react-ui-kit'; export default function App() { return ( Unbound ); }

Radius

By using rippleRadius property you can modify the radius of the ripple. The numeric value is expressed in pixels, for example: rippleRadius={10}

import React from 'react'; import { MDBBtn } from 'mdb-react-ui-kit'; export default function App() { return ( <> Default 10 25 50 ); }
