- API

Import

import { MDBAutocomplete } from 'mdb-react-ui-kit';

Properties

MDBAutocomplete

Name Type Default Description Example
className String '' Add custom class to the MDBAutocomplete <MDBAutocomplete className="class" />
customContent Element - Custom HTML template that will be displayed at the end of the component dropdown <MDBAutocomplete name="sampleName" label="Example label" id="inputExample" />
dataFilter Function - Function that returns filtered data to the component <MDBAutocomplete value="Example value" label="Example label" id="inputExample" />
displayValue Function - Function executed for complex search results, to get value to display in the results list <MDBAutocomplete label="Example label" id="inputExample" />
inputValue String - Stores value of the MDBAutocomplete input <MDBAutocomplete label="Example label" id="inputExample" />
itemContent Function - Function that returns custom template for result item <MDBAutocomplete label="Example label" labelId="exampleLabel" id="checkboxExample" />
setInputValue String - Set value of the MDBAutocomplete input <MDBAutocomplete label="Example label" labelId="exampleLabel" labelClass="test-class" id="checkboxExample" />
threshold Number 0 Minimum input length to start search query <MDBAutocomplete wrapperClass="custom-class" label="Example label" id="checkboxExample" />
tag String 'div' Defines a tag type for the MDBAutocomplete <MDBAutocomplete tag='span' />
