React Bootstrap Link

React Link - Bootstrap 4 & Material Design

Link is a component which allows to use link from react-router-dom in the mdbreact pacakge. This component was created to replace a component MDBNavLink from react-router-dom in pills, tabs and other component when is no nessesary to use nav-link from react-router-dom. You can easily replace MDBNavLink with MDBLink in your components like pills, tabs etc. and nothing will be break.
