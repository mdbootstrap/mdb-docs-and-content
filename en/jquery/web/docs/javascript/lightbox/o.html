LightBox MDB Pro component

Bootstrap lightbox

A Bootstrap lightbox is a group of images combined in one responsive gallery. Elements are grouped in a thumbnail grid, which can be displayed as a slideshow.

Basic example

Example code:

 // MDB Lightbox Init $(function () { $("#mdb-lightbox-ui").load("mdb-addons/mdb-lightbox-ui.html"); });

Initialization required

To use our Lightbox you have to initialize it in your project with the code below.

Initialization code:

// MDB Lightbox Init $(function () { $("#mdb-lightbox-ui").load("mdb-addons/mdb-lightbox-ui.html"); });

Predefined image size

A MDB LightBox uses the same image in two different sizes: thumbnail and large.

Such an approach let you increase the performance of your website.

// MDB Lightbox Init $(function () { $("#mdb-lightbox-ui").load("mdb-addons/mdb-lightbox-ui.html"); });

Important:

1. The data-size attribute should equal the size of the Large Image

2. The Thumbnail and Large images should have the same ratio

3. The Thumbnail should be 544px wide

4. The Large image shouldn't be wider than 1800px

 // MDB Lightbox Init $(function () { $("#mdb-lightbox-ui").load("mdb-addons/mdb-lightbox-ui.html"); });
