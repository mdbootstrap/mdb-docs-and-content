Datatables

Bootstrap 4 datatables integration

MDBootstrap integration with a very popular plugin enhances the possibilities of standard tables.

To start working with DataTables, please see "Getting Started" tab on this page.

Note: Datatables integration is available from version 4.5.7 (released 16.07.2018).

To use DataTables, you have to include the path to DataTable, then select CSS and JavaScript. The files are located in the addons folder.
