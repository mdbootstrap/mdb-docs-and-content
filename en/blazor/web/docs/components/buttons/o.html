Waves effect with outline buttons

MDB automatically adds waves effect to each button component. However, in outline buttons, it's barely visible due to lack of background.

That's why to outline buttons we add additional class .waves-effect to make waves darker and more intense. Use Parameter WavesEffect="true", that's the default.

Create block level buttons—those that span the full width of a parent—by adding the Parameter Block="true".

Active state

Buttons will appear pressed (with a darker background, darker border, and inset shadow) when active. There’s no need to add a class to <button>s as they use a pseudo-class. However, you can still force the same active appearance by setting the parameter Active="true" (and include the aria-pressed="true" attribute) should you need to replicate the state programmatically.
