Vue Bootstrap Hotkey MDB Pro component

Vue Hotkey - Bootstrap 4 & Material Design

The mdb-hotkey is a custom directive which allows to set callbacks for key combinations.

Live Preview

Basic usage

Step 1: Import the directive from 'mdbvue'

Step 2: Add mdbHotkey to the directives object

Step 3: Attach the directive to the element on which you wish to listen for events and pass a keymap as the value.

Options

Name Description Example
:window Adds the event listener to the window instead of the element <div v-mdb-hotkey:window="keymap" ></div>
:document Adds the event listener to the document instead of the element <div v-mdb-hotkey:document="keymap" ></div>

Modifiers

Name Description Example
.stop Calls stopPropagation method on the keydown event <div v-mdb-hotkey.stop="keymap" ></div>
.prevent Calls preventDefault method on the keydown event <div v-mdb-hotkey.prevent="keymap" ></div>
