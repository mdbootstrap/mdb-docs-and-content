Bootstrap Time Picker Premium component

Vue Timepicker is a component that allows user to select in a clean and direct manner. The components accepts many props, making it highly customizable.

Basic example

Options

Name Default Type Description
id timePickerInput String sets inputs id
placeholder Select time String sets placeholder for time picker input
color primary-color String applies specified color to the control
label null String sets label for time picker input
hoursFormat 12 Number defines hours format, available: 12 and 24 hrs format
