Carousel - API

In this section you will find advanced information about the Carousel component. You will learn which modules are required in this component, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Import statement

Rendering content

The only property required for mdbCarousel to work is items, and it accepts two datatypes: array and number. When given an array, it will render a simple image/video carousel basing on passed data, while the number will resolve in this quantity of named slots. Slots are named simply after indexes (starting at 1). Each of the slides has to be wrapped in the template tag with the reference to the corresponding slot.

API Reference: Properties

Name Type Default Description Example
items - required Array, Number Specifies the number (and content if it's an array) of items <mdb-carousel :items="3">
interval Number If given a value, the component will change slide after this time (in milliseconds) <mdb-carousel :items="3" :interval="8000">
tag String div Specifies the component's wrapper tag <mdb-carousel :items="3" tag="span">
slide Boolean false Changes the animation to sliding effect <mdb-carousel :items="3" slide>
controlls Boolean false If set to true, control arrows will be shown <mdb-carousel :items="3" controlls>
controllIcons Array Sets custom icons for controlls <mdb-carousel :items="3" :controllIcons="['fas fa-arrow-circle-left fa-2x text-white', 'fas fa-arrow-circle-right fa-2x text-white']">
indicators Boolean false If set to true, indicators will be shown <mdb-carousel :items="3" indicators>
multi Boolean false Changes the carousel's style to multi-item <mdb-carousel :items="3" multi>
one Boolean false Makes the multi-item carousel slide one item per time <mdb-carousel :items="3" multi one>
thumbnails Boolean false Changes the carousel's style to thumbnails <mdb-carousel :items="3" thumbnails>
testimonial Boolean false Changes the carousel's style to testimonial <mdb-carousel :items="3" testimonial>
touch Boolean false Adds swipe right/left events to carousel <mdb-carousel :items="3" touch>
floating Boolean true Sets controlls' style to floating <mdb-carousel :items="3" controlls :floating="false">
leftIcon String chevron-left Changes the icon of the left control button <mdb-carousel :items="3" controlls leftIcon="paper-plane">
rightIcon String chevron-left Changes the icon of the right control button <mdb-carousel :items="3" controlls rightIcon="paper-plane">
navClass String Adds new class to the controlls <mdb-carousel :items="3" controlls navClass="custom_class">
navColor String primary Changes the color of the controlls <mdb-carousel :items="3" controlls navColor="secondary">
indicatorsColor String primary Changes the color of the indicators <mdb-carousel :items="3" controlls indicatorsColor="secondary">
thumbnailWidth Number 100 Changes the width of thumbnails <mdb-carousel :items="3" thumbnails :thumbnailsWidth="50">

- property available only for PRO users.
